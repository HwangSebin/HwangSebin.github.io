UI/UX comment 기존 웹사이트와 모바일사이트를 따로 운영하는 방식에서 관리의 편의를 위하여 반응형 사이트로 리뉴얼 하였으며, 미국에서만 접속 가능한 소모품 웹 Store을 오픈하였다. 검색 포털에서 검색시 홈페이지로 접근이 용이 하도록 SEO에 최적화 되어 있는 워드프레스를 기반으로 제작하였다. 기존 텍스트 중점인 사이트에서 많은 콘텐츠에 집중하기 위하여 이미지중심의 레이아웃으로 기획 하였다.
MAIN 기존 GNB와 동일한 스타일로 진행하였으며, 사이트 내 검색 및 이후 추가될 불어 사이트를 염두하여 사이트 선택버튼을 Drop down으로 변경 하였다. 메인 슬라이드 아래 이벤트 배너를 배치하여 전시회 참가 및 이벤트를 홍보시킬 수 있도록 하었으며, 신제품 홍보 및 뉴스 를 상단으로 배치하여 기존 레이아웃보다 새로운 제품과 소식이 노출 될 수 있도록 하였다.
PRODUCT PAGE 기존 탭메뉴 형식으로 제작 되어 정보가 가려져 있어 정보를 찾기 어려운 단점을 보완하여 모든 정보를 오픈 시켜 사용자가 자연스럽게 데이터를 볼 수 있도록 하였고, 네비게이션을 이용하여 원하는 정보를 찾을 수 있도록 유도 하였다.
REQUEST A QUOTE 기존 제품의 다중 선택이 불가 했던 방식과 상세한 견적을 받기에 어려웠던 형식에서 다중선택 및 소모품까지 견적을 받을 수 있도록 기획하였다. 소비자 및 견적을 내줘야하는 직원의 편의를 고려하여 제작되었다.
UI/UX comment 홈페이지 리뉴얼 후 CELENA-X 론칭 되어 페이지 추가 작업을 진행하였다.
전체적으로 CELENA-S의 레이아웃을 중점 으로 진행하였으나 프로그램의 Application 부분이 추가 되었으며, 세포 이미지를 활용하여 배너형식의 스타일로 사용자가 원하는 App을 쉽게 찾을 수 있도록 이미지 카드 형식으로 제작 하였다.
dev comment Bootstrap을 활용하였으며, 기존 CELENA-S 의 페이지의 소스를 활용하여 작업하였다.
Applications 페이지는 추후 컨텐츠가 추가가 될것을 고려하여 간단한 css 및 tag를 사용하여 제작되었다. SEO를 고려하여 이미지에 alt 값 및 title을 추가 하여 작업하였다. 하단 리소스들은 include 방식으로 구성되어있으므로, jsp문서를 활용하여 작업하였다.
