WEB design & WEB publishing 15. 09 ~ 16. 04
PROJECT 마플 2.0 리뉴얼 디자인 ( 15. 09 ~ 16.04 )
마플 2.0 상품상세 페이지 합성작업 ( 16. 03 ~ 16.04 )
TYPE WEB GUI design
TOOL Photoshop CC
Illustrator cs3
GUI comment 온라인 커스텀 사이트 '마플' 리뉴얼 디자인 작업, 기존의 무거웠던 UI에서 벗어나 화이트, 블랙을 사용하여 가벼운 느낌의 마플 만들기툴의 UI 만들었으며, 사용자가 컨텐츠에 집중할 수 있도록 무채색의 컬러로 디자인 하였다.
CUstom Tool 블랙&화이트 를 사용하여 최대한 사용자가 커스텀하는것에 집중하도록 설계 하였으며, 이미지, 텍스트 추가시 윈도우에 depts를 주어 유동적으로 움직이는 영역이라는것을 표현하였다.
Review 사용자들의 컨첸트가 들어가야할 공간이기에 최대한 색은 배제 시키고, 사용자들의 리뷰가 돋보일 수 있도록 작업하였다.
Cart 총 주문 금액의 주문하기 버튼의 힘을 주기 위하여 버튼에 입체감을 주었으며, 총 주문 금액 영역을 영수증으로 표현하였다. 상품리스트는 카드형으로 작업하였으며, 단체 주문이 많은 특성상 사이즈, 가격, 할인가격등을 상세히 표현하였다.
HELP 각 다른 성질의 데이터들의 영역구분을 위하여 그룹핑 처리 하였으며, 아이콘을 사용하여 사용자가 상세히 보지 않아도 어떠한 내용의 데이터인지 쉽게 알 수 있도록 제작 하였다.
News / event 공지 사항 및 이벤트를 탭으로 사용하여 사용자가 두 메뉴의 컨텐츠를 혼동없이 볼 수 있도록 설계하였다.
Mypage 탭형식의 LNB를 사용하였으며, 사용자가 한눈에 데이터를 확인 할 수 있도록 설계하였다.
