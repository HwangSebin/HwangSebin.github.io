services 다달이 정기적으로 나가는 금액에 관해 불편함을 느끼고, 조금의 돈이라도 허투로 쓰고싶지 않은 유저가 사용하는 줄줄 새는 온라인 결제 금액을 한 번에 잡아주는 서비스 LEAKY
User Behavior 정기결제 서비스 결제 전 서비스 종류를 정한다 → 서비스를 저렴하게 쓰고 싶어 이것저것 비교를 한다 → 정기적으로 이걸 쓰게 될까? → 정기적으로 쓴다면 매월 자동결제를 하자 → 한달에 0.99 달러씩 나가는 정도면 괜찮네 → 언제 까지 사용할지 메모장에 적어놓자 → 결제 예정일을 적어놓는다 정기결제 서비스 결제 카드 결제를 한다 → 첫 결제가 난다 → 매월 이 날에 결제가 발생한다는 것을 알게 된다 → 노트나 캘린더로 정기결제 할부 시작일을 기록해둔다 정기결제 서비스 사용 중 일주일 정도 열심히 사용하다 점점 서비스 접속이 뜸해진다 → 결제 내역이 오지만 어떤 내역인지 확인 하고 딱히 행동을 취하지 않는다 → 해당 서비스의 실시간 사용시간을 생각해본다 → 서비스를 다시 열심히 사용해본다 → 해지를 해볼까 생각한다 → 해지를 하려다 포기한다 → 망각 후 다시 결제가 일어난다
