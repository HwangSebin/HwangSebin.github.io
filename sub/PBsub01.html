Web Publishing
Dr.jart Renewal
본 프로젝트는 Dr.jart 홈페이지 리뉴얼 프로젝트로 Bootstrap3을 사용하여 반응형 웹페이지를 만들었으며, Ajax 통신을 이용하여 실시간으로 클라이언트와 서버가 통신할 수 있는 환경을 구축하였다. 회원가입 시 Javascript 정규표현식을 사용하여 문자열을 검열할 수 있으며, Bootstrap의 Validation states로 클라이언트에게 즉각적인 UX를 구현하였다. 우편번호 검색 시 가상 DB에서 데이터를 가지고 올 수 있도록 하였다.
리뷰 페이지에선 jQuery Plugin 인 SmootScroll를 사용하여 페이지 이동 및 ScrollTop 버튼을 누를 경우 부드러운 화면 이동을 구현하였고,
Css의 transform를 사용하여 마우스 오버 시 이미지 확대 기능을 추가하였다. ImgLiquid를 활용하여 각각 다른 이미지의 크기를 어느 기기에서나 같은 사이즈의 이미지 크기로 보이게 구현하였다.
* 개인 작업범위만 인덱스 페이지에 구현되여있음.
